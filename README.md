# Treinamentos Edeconsil

Módulo standalone da plataforma de treinamentos internos da Edeconsil.

## Tecnologias
- React + Vite + TypeScript
- GitHub Pages (deploy automático)

## Desenvolvimento local
```
npm install
npm run dev
```

## Deploy
Push para a branch main dispara o deploy automático via GitHub Actions.

URL: https://pabloHenrique97.github.io/treinamentos-edeconsil/
